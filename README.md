# Electric Vehicle Data Analysis
Analysis of electric vehicle adoption and trends using real-world data

## Overview
This project analyzes global electric vehicle (EV) adoption trends using real-world data to understand growth patterns, regional differences, and key drivers of adoption.

## Data Source
Public electric vehicle population dataset (Kaggle).

## Tools Used
- Excel
- Data analysis & visualization techniques
- Reporting and presentation

## Key Insights
- EV registrations increased significantly from 2011 to 2024
- Battery Electric Vehicles (BEVs) dominate the market
- Major cities such as Seattle lead in EV adoption
- Average EV range has improved substantially over time

## Skills Demonstrated
- Data cleaning and preparation
- Exploratory data analysis
- Data visualization
- Insight communication and reporting
